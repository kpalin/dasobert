# dasobert
Dasobert is a Java DAS client library. It is based on BioJava. Currently it supports the DAS/1 spec and there are some bits in there that will provide support for DAS/2 as well. 

This git repository is was imported from SVN repository http://www.derkholm.net/svn/repos/dasobert/trunk/ of @andreasprlic by @kpalin
